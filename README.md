# Sorting Algorithms Test

A program for testing different sorting algorithms.

Python Version: **3.5**

Supported OS: **OSX / Linux / Windows**

## Algorithms Supported
* Some Sort (I don't know what algorithm is this)
* Insertion Sort
* Selection Sort
* Bubble Sort
* Merge Sort
* Bogo Sort

## Arrays Supported
* Consecutive array
* Random array
* Few unique array
* Nearly sorted array
* Reversed array
